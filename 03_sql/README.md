# Домашнее задание 02: Разработка REST API сервиса

---

REST API сервис для системы управления медицинскими записями.

**Вариант:** 20  
**Автор:** Занозин Александр, М8О-107СВ-25

---

## Описание

Система предназначена для управления медицинскими записями пациентов, включая:
- Регистрацию пользователей и пациентов
- Ведение медицинских карт
- Поиск по пользователям и пациентам
- Управление медицинскими записями

---

## Быстрый старт

### Требования

| Компонент | Версия |
|-----------|--------|
| CMake | ≥ 3.15 |
| C++ Compiler | ≥ C++17 |
| POCO | ≥ 1.12 |
| Docker | ≥ 20.10 |
| Docker Compose | ≥ 2.0 |

### Сборка
```bash
make build
```

### Запуск тестов

```bash
make test
```

### Сборка + тесты
```bash
make build-tests
```

### Очистка
```bash
make clean
```

### Открыть Swagger UI
```
http://localhost:8081
```
